Hybrid Deep Learning Framework for Automated Lung Cancer Diagnosis

An advanced hybrid deep learning model combining CNNs and Vision Transformers is proposed for early lung cancer detection using CT scans. By integrating InceptionNeXt blocks with grid and block attention mechanisms, the model effectively captures both fine and large-scale features, enabling accurate classification of lung nodules and cancer subtypes. Trained on public datasets, the model achieves up to 99.54% accuracy while remaining lightweight and efficient, making it a promising solution for improved clinical diagnosis.
